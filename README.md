# Desafio sobre Design Patterns - Sistema de Checkout em Java

Este projeto demonstra o uso de três padrões de projeto em Java:

## 🧠 Padrões utilizados

### Singleton
Usado para manter uma instância única de configuração global (AppConfig).

### Strategy
Usado para aplicar diferentes regras de desconto (VIP, Promo, Nenhum desconto).

### Facade
Simplifica o processo de compra através de uma interface única (CheckoutFacade).

---

## 🚀 Como executar

1. Abra o projeto no IntelliJ IDEA
2. Execute a classe Main.java
3. Veja o resultado no console

---

## 💡 Objetivo

Praticar Design Patterns em Java puro, simulando um sistema de checkout simples.
